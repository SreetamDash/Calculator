# 🧮 Calculator — Day 1 of #10DaysOfWebDev

A simple and clean calculator built with HTML, CSS, and JavaScript.

---

## 🔗 Links

- **Live Demo:** [https://calculatorweb001.netlify.app/]
- **GitHub Repo:** [https://github.com/SreetamDash/Calculator]

---

## ✨ Features

- Basic operations — Add, Subtract, Multiply, Divide
- Percentage calculation
- Delete last digit (Back button)
- Clear display (C button)
- Error handling for invalid inputs
- Glassmorphism UI design

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript (Vanilla)

---

## 📁 Project Structure

```
calculator/
├── index.html
├── style.css
└── script.js
```

---

## 🚀 How to Run

1. Clone the repo or download the files
2. Open `index.html` in your browser
3. That's it — no setup needed!

---

## 💡 What I Learned

- CSS Grid for button layout
- Glassmorphism styling using `backdrop-filter`
- Using `eval()` for expression calculation with `try/catch` for error handling
- `slice(0, -1)` to delete the last character
